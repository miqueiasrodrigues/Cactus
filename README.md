# Cactus
O Cactus é uma aplicativo desenvolvido para o sistema operacional Android, construído utilizando a linguagem de programação Dart e o Framework Flutter que possibilita realizar uma conexão de forma indireta com o microcontrolador ESP32, por meio do banco de dados Firebase. A aplicação tem a função de enviar informações para o ESP32 ou coletar e mostrar as informações vindas dele.
